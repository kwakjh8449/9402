# 장부 영수증 앱

영수증 OCR 스캔 + 수기 입력 + 품목 가격 비교 + JSON 백업/복구가 되는 개인용 웹앱이야.

## 기능
- 영수증 사진 OCR 인식
- 영수증이 없을 때 직접 구매내역 입력
- 상품명 자동 통합(예: 서울우유 1L -> 우유)
- 품목 검색
- 날짜별 가격 비교
- localStorage 자동 저장
- JSON 파일 백업/복구

## 실행
```bash
npm install
npm run dev
```

## Vercel 배포
1. 이 폴더를 GitHub에 올리기

4. Deploy

## 참고
- OCR은 사진 선명도에 따라 정확도가 달라져.
- 현재 저장은 브라우저 기준이야.
- 친구에게 ZIP을 보내도 되고, 각자 배포해서 써도 돼.
